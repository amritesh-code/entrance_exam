# Adira Entrance (React + Vite, Modular + Local Tailwind)

A componentized React port of your AI Voice Entrance Exam, with local Tailwind CSS (no CDN).

## Quick start

```bash
# 1) Install deps
npm install

# 2) Dev server
npm run dev

# 3) Production build
npm run build
npm run preview
```

Open the dev URL (usually http://localhost:5173). Use Chrome/Edge for mic permission + Web Speech API.
